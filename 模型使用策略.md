# 模型使用策略

## 🎯 核心原则
**智能匹配，按需切换** - 根据任务复杂度自动选择最合适的模型

## 🤖 可用模型

### 1. `deepseek/deepseek-chat` (默认)
- **特点:** 快速响应、成本效益高、适合常规任务
- **适用场景:** 日常信息收集、文本处理、简单问答
- **调用方式:** 默认自动使用

### 2. `deepseek/deepseek-reasoner` (推理增强)
- **特点:** 深度推理、复杂分析、策略规划
- **适用场景:** 技术趋势分析、学习路径规划、复杂问题解决
- **调用方式:** 手动指定或自动触发

## 🔧 自动切换策略

### 基于任务类型的切换
```python
def select_model(task_type):
    # 使用chat模型的场景
    chat_tasks = [
        "daily_collection",      # 日常收集
        "text_formatting",       # 文本格式化
        "simple_classification", # 简单分类
        "status_check",          # 状态检查
        "log_analysis"           # 日志分析
    ]
    
    # 使用reasoner模型的场景
    reasoner_tasks = [
        "trend_analysis",        # 趋势分析
        "strategy_planning",     # 策略规划
        "complex_problem",       # 复杂问题
        "learning_path",         # 学习路径
        "technical_insight"      # 技术洞察
    ]
    
    if task_type in reasoner_tasks:
        return "deepseek/deepseek-reasoner"
    else:
        return "deepseek/deepseek-chat"
```

### 基于内容复杂度的切换
```python
def select_model_by_complexity(content):
    # 计算内容复杂度
    complexity_score = calculate_complexity(content)
    
    # 切换阈值
    if complexity_score > 0.7:  # 高复杂度
        return "deepseek/deepseek-reasoner"
    else:  # 中低复杂度
        return "deepseek/deepseek-chat"
```
